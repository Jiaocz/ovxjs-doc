# 通用配置
本页将介绍项目中通用的预设配置项及参考文档。

## `package.json`
文件位于项目根目录中，其中配置了项目的各项依赖包，以及一下几个脚本（script）

- `dev`: 启动调试服务器
- `build`: 构建项目
- `preview`: 打开预览服务器预览构建完成的项目
- `lint`: 进行语法检查
- `lint:fix`: 语法检查并修复
- `type-check`: TypeScript 语法检查

参考配置文档：[package.json - npmjs docs](https://docs.npmjs.com/cli/v7/configuring-npm/package-json)
